## Yerlesik Agent — Kur ve Kullan, Sifir Yapilandirma

CoraCowork tam bir AI Agent motoru ile birlikte gelir. CLI Agent'lari ayri olarak kurmanizi gerektiren araçlarin aksine, **CoraCowork kurdugunuz anda çalisir**.

- **Kurulacak CLI araçlari yok** — Agent motoru yerlesiktir
- **Karmasik kurulum yok** — baslamak için herhangi bir API anahtari yapistirin

## Çoklu Agent Modu — Zaten CLI Agent'lariniz var mi? Getirin

Zaten Claude Code, Codex, Hermes Agent veya OpenClaw kullaniyorsaniz, CoraCowork bunlari otomatik olarak algilar ve hepsiyle Cowork yapmaniza izin verir — yerlesik Agent ile birlikte.

**Desteklenen Agents:** Yerlesik Agent (sifir kurulum) • Claude Code • Codex • Qwen Code • Goose AI • OpenClaw • Augment Code • CodeBuddy • Kimi CLI • OpenCode • Factory Droid • GitHub Copilot • Qoder CLI • Mistral Vibe • Nanobot • Cora CLI (corars, CoraCowork ile birlikte dagitilan Rust tabanli arka uç servisi) • Snow CLI • Hermes Agent • Cursor Agent ve daha fazlasi

<p align="center">
  <img src="../../resources/multi-agent??openclaw.gif" alt="Multi-Agent Cowork" width="800">
</p>

- **Otomatik Algilama** — kurulu CLI araçlarini otomatik olarak tanir
- **Birlesik Arayüz** — tüm AI Agent'lariniz için tek bir Cowork platformu
- **Paralel Oturumlar** — bagimsiz baglam ile birden fazla Agent'i ayni anda çalistirin
- **MCP Birlesik Yönetimi** — MCP (Model Context Protocol) araçlarini bir kez yapilandirin, tüm Agent'lara otomatik olarak senkronize edilir — her Agent'i ayri ayri yapilandirmaniza gerek yok
- **YOLO Mode** (tüm Agent eylemlerini manuel onay olmadan otomatik onayla) / **Tam Otomatik Mod** — tek tikla izin istemlerini atlayin; tüm Agent'lar gözetimsiz çalistirma için tam otomatik modu destekler

### Team Mode — Koordineli Çoklu Agent Isbirligi

Birden fazla AI Agent'i organize bir ekip olarak çalistirin: bir **Leader** Agent talimatlarinizi alir, bunlari alt görevlere böler ve yerlesik Team MCP Server araciligiyla **Teammate** Agent'lara devreder. Teammate'ler paralel olarak çalisir, asenkron posta kutusu üzerinden sonuçlari paylasir ve paylasilan görev panosuna yazar.

<p align="center">
  <img src="../../resources/CoraCowork_team.gif" alt="Team Mode overview" width="800">
</p>

- **Paralel çoklu agent çalistirma** — Leader görevleri alt görevlere bölerek paralel çalisan Teammate Agent'lara devreder; her Teammate ACP (Agent Communication Protocol, CoraCowork'nin çoklu agent koordinasyon katmani), Gemini veya Corars araciligiyla kendi modelini kullanir
- **Leader orkestrasyonu** — Leader atar, izler ve sonuçlari toplar; desteklenen arka uçlar: Claude Code, Codex, Hermes Agent, Gemini, Snow CLI ve Cora CLI
- **Ekip izole çalisma alani** — tüm Agent'lar ayni klasörü paylasir; her birinin bekleyen onaylar için kenar çubugu rozeti ile kendi izin onay iletisim kutusu vardir

### **Zamanlanmis Görevler — Otomatik Pilotta Cowork**

_Bir kez ayarlayin, AI Agent zamanlamaya göre otomatik olarak çalisir — gerçek 7/24 gözetimsiz islem._

- **Dogal Dil** — Agent'a ne yapacagini söyleyin, sadece sohbet ediyormus gibi
- **Üç zamanlama modu** — standart cron ifadesi (saat dilimi destegiyle), sabit aralik (her N dakika/saat) veya tek seferlik tetikleme
- **AI tarafindan olusturulan görevler** — Agent'lar bir konusma sirasinda özerk olarak zamanlanmis görevler olusturabilir
- **Kullanim Senaryolari:** zamanlanmis veri toplama, rapor olusturma, dosya organizasyonu, hatirlaticilar

<p align="center">
  <img src="../../resources/alart-task.png" alt="Scheduled Tasks" width="800">
</p>

<details>
<summary><strong>?? Zamanlanmis Görev Detaylarini Görüntüle ??</strong></summary>

<br>

**Zamanlama modlari:**

- `Cron ifadesi` — saat dilimi destegiyle standart 5 alan cron (örn. `0 9 * * 1`, `Asia/Shanghai`)
- `Her N dakika/saat` — sabit aralik, örn. her 30 dakikada bir çalistir
- `Tek seferlik` — belirtilen tarih/saatte bir kez tetikle, sonra otomatik olarak devre disi birak

**Çalistirma modlari:**

- `Mevcut konusmada devam et` — bagli konusmaya ekler, AI tam baglam geçmisini korur
- `Her seferinde yeni konusma olustur` — her tetiklemede yeni bir oturum açar, bagimsiz periyodik raporlar için idealdir

**Diger yetenekler:**

- **Konusmaya Bagli** — Her zamanlanmis görev bir konusmaya baglidir, baglami ve geçmisi korur
- **Otomatik Yürütme** — Görevler zamanlanmis saatlerde otomatik olarak çalisir, konusmaya mesaj gönderir
- **Kolay Yönetim** — Zamanlanmis görevleri istediginiz zaman olusturun, degistirin, etkinlestirin/devre disi birakin, silin ve görüntüleyin
- **Uyanik kal** — CoraCowork görevler aktifken sistem uyku moduna geçmeyi otomatik olarak önler ve uyandiktan sonra kaçirilan tetikleyicileri algilar
- **Gelismis yapilandirma** — her görevin kendi model, çalisma alani dizini ve akil yürütme eforu ayarlari olabilir

**Gerçek Dünya Örnekleri:**

- Günlük hava durumu raporu olusturma
- Haftalik satis verisi toplama
- Aylik yedek dosya organizasyonu
- Özel hatirlatici bildirimleri

</details>

| Boyut           | Claude Cowork | CoraCowork                                            |
| :-------------- | :------------ | :------------------------------------------------ |

| Etkilesim       | Masaüstü GUI  | Masaüstü GUI + WebUI + Telegram / Lark / DingTalk |

| Maliyet         | $100/ay       | Ücretsiz ve Açik Kaynak                           |

Derin AI Ofis Senaryosu Destegi:

- **Dosya Yönetimi**: Yerel klasörleri akillica organize edin ve tek tikla toplu yeniden adlandirin.
- **Veri Isleme**: Excel raporlarini derinlemesine analiz edin ve otomatik olarak güzellestirin.
- **Belge Olusturma**: PPT, Word ve Markdown belgelerini otomatik olarak yazin ve biçimlendirin.
- **Aninda Önizleme**: Yerlesik 10+ format önizleme panelleri, AI isbirligi sonuçlari aninda görünür.

## Hizli Soru-Cevap

<details>
<summary><strong>S: Önce Gemini CLI veya Claude Code'u kurmam gerekiyor mu?</strong></summary>
C: <strong>Hayir.</strong> CoraCowork kurulumdan hemen sonra çalisan yerlesik bir AI Agent'a sahiptir. Baslamak için herhangi bir API anahtari girmeniz yeterli. Ayrica Claude Code veya Gemini CLI gibi CLI araçlariniz da kuruluysa, CoraCowork bunlari otomatik olarak algilayacak ve daha fazla yetenek için entegre edecektir.
</details>

<details>
<summary><strong>S: CoraCowork ile ne yapabilirim?</strong></summary>
C: CoraCowork sizin <strong>özel Cowork çalisma alaninizdir</strong>. Yerlesik Agent klasörleri toplu olarak organize edebilir, Excel verilerini isleyebilir, belgeler olusturabilir, web'de arama yapabilir ve görüntüler olusturabilir. Çoklu Agent Modu ile ayni arayüz üzerinden Claude Code, Codex ve diger güçlü CLI Agent'larindan da yararlanabilirsiniz.
</details>

<details>
<summary><strong>S: Ücretsiz mi?</strong></summary>
C: CoraCowork tamamen ücretsiz ve açik kaynaklidir. Yalnizca seçtiginiz saglayicinin API kullanimi için ödeme yaparsiniz ve tercih ettiginiz herhangi bir saglayicidan API anahtarlari kullanabilirsiniz.
</details>

<details>
<summary><strong>S: Verilerim güvende mi?</strong></summary>
C: Tüm veriler yerel olarak bir SQLite veritabaninda saklanir. Hiçbir sey herhangi bir sunucuya yüklenmez.
</details>
